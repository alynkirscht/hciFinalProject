Instructions

To run website using flask:
- run flask app:
    flask --app app --debug run
- copy http://127.0.0.1:5000 and open on browser and add /login
or copy this link:
    http://127.0.0.1:5000/login

To login use:
    -Any music platform
    - Username: a
    - Password: 1

For languages you can choose:
    - Spanish
    - Hindi
    - Spanish and Hindi

    - So on search bar type: Dura
